# TriFactor Legal Pages

This repository hosts the public legal pages for TriFactor.

It is intentionally separate from the main game repository and contains only the files that need to be publicly accessible, such as the privacy policy used for the Google Play listing.

## Contents

- `privacy-policy.html`: public privacy policy page for TriFactor

## Purpose

This repository exists so that TriFactor can publish a public privacy policy URL without exposing the game's source code repository.
